Drawing paper.js and jsxgraph

Vector Drawing

Vector Drawing

Instructions:

Click anywhere on the protractor, drag the mouse, and release to create a vector. Note, grading on edX requires the vector to start from the center of the protractor.

Drawing and interpreting vectors are fundamental to many physics concepts. Paper.js allows students to draw directly to screen using their mouse. Inspired by problems in force diagrams.

Force Diagrams

Force Diagram

Instructions:

Use the buttons to the right of the canvas to insert vectors. Then drag them to their appropriate location.

Force Diagrams are used throughout physics courses to analyze forces acting on an object. This example allows students to arrange vectors vectors and get graded feedback on their configuration. Any image can be loaded to the background.

Open-Source Javascript Packages

JSXgraph

From their website: JSXGraph is a cross-browser library for interactive geometry, function plotting, charting, and data visualization in a web browser. It is implemented completely in JavaScript, does not rely on any other library, and uses SVG, VML, or canvas. JSXGraph is easy to embed and has a small footprint: less than 100 KByte if embedded in a web page. No plug-ins are required! Special care has been taken to optimize the performance.

Paper JS

From their website: Paper.js is an open source vector graphics scripting framework that runs on top of the HTML5 Canvas. It offers a clean Scene Graph / Document Object Model and a lot of powerful functionality to create and work with vector graphics and bezier curves, all neatly wrapped up in a well designed, consistent and clean programming interface.
